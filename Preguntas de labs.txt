¿Qué es SQL injection y cómo se puede prevenir?
Es ingresar comandos de SQL a través de entradas que el programa permite. Para prevenirlo es muy recomendable desinfectar las entradas.

¿En qué consiste el control de acceso basado en roles?
Es un enfoque en los sistemas que permite a usuarios accesar información dependiendo de su nivel de autorización. Cada usuario tiene un rol y cada rol tiene distintos permisos.

Investiguen y describan 2 sistemas, uno que aplique RBAC y uno que no. Realicen un análisis de las ventajas y desventajas de cada uno con respecto al control de acceso.
WordPress es un sistema que usa RBAC. Existen autores, editores, suscriptores y admins. 
Drupal es un sistema que no usa RBAC. Los permisos son otorgados individualmente a los usuarios.
Ventajas de usar RBAC:
Mejor seguridad
Escalabilidad
Gestión de permisos
Desventajas de usar RBAC:
Complejidad
Falta de gestión granular de usuarios

Para que el sitio fuera más accesible, decidí seguir los lineamientos de uxdesign. Agregué contraste en los colores, focus indicators para los tabs y en vez de usar placeholders puse texto arriba. De igual manera, el logo tiene texto alternativo que describe lo que es. Si agregamos más imágenes, agregaremos más texto alternativo. También usé headings para los títulos y texto para definir jerarquías. 
¿Por qué es importante la accesibilidad?
La accesibilidad es importante para que no haya barreras entre los usuarios y el software. Al agregar opciones para usuarios con capacidades diferentes, los incluyes y le das más valor a la aplicación.
¿Cuáles son los lineamientos de accesibilidad que consideras más importantes? ¿Por qué?
Considero que los lineamientos de accesibilidad más importantes son los del contraste, markup correcto y usar alt text para las imágenes. Esto permite que usuarios con visión baja puedan entender más fácilmente la página.
Describe 3 cambios que harás en tu proyecto para hacerlo más accesible.
Haré gráficas accesibles, permitiré usar tab para navegación y haré pruebas con una persona con visión baja para que me diga las áreas de oportunidad
